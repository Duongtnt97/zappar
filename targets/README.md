# Hướng dẫn tạo Image Target (.zpt)

## Cách tạo file .zpt:

1. **Truy cập Zappar Studio**: https://studio.zappar.com/
2. **Tạo project mới** → chọn "Image Tracking"
3. **Upload ảnh target** (nên là ảnh có nhiều chi tiết, contrast cao)
4. **Export** → Download file `.zpt`
5. **Đặt file vào**: `public/targets/your-target.zpt`

## Yêu cầu ảnh target tốt:
- ✅ Nhiều chi tiết, góc cạnh
- ✅ Contrast cao (đen/trắng rõ ràng)
- ✅ Không quá đối xứng
- ❌ Tránh ảnh mờ, đơn sắc
- ❌ Tránh ảnh có nhiều text nhỏ

## Kích thước khuyến nghị:
- Ảnh gốc: tối thiểu 512x512px
- In ra: A5 (148x210mm) hoặc lớn hơn để dễ nhận diện
